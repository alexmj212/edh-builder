## Critical Anti-Patterns (do NOT repeat these)

- Using `[derivedBooleanFlag, debouncedKey]` as combined effect deps when the boolean is itself derived from state that flows into the debouncedKey. Causes a double-fire on the boolean's transition. → Fold the boolean into the key by returning `null` when the guard would have returned early.
- Mounting a component that fires network I/O on mount while an upstream hydration is still in flight. → Gate the parent on the hydration's `loading` flag and render a skeleton.
- Relying solely on store-level AbortController to dedupe StrictMode mount effects. The abort cancels response handling but not the HTTP dispatch — bytes are already on the wire. → Add a ref-keyed dedupe OR use React Query / TanStack Query which handles this natively.
