## Sampling Rate

- **After every task commit:** Run `npm run typecheck` (fast — catches Card-parity regressions) + relevant-module vitest
- **After every plan wave:** Run `npm run test -- --run` (full unit) + affected Playwright spec
- **Before `/gsd-verify-work`:** Full suite must be green, including `npm run e2e`
- **Max feedback latency:** ~30 seconds per task, ~90 seconds per wave
