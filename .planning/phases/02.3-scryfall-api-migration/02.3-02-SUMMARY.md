## Decisions Made

All locked decisions from plan 02.3-01 SUMMARY honored:

- **D-04 promise-race wrapper:** single `abortable<T>(p, signal?)` helper in `src/lib/scryfall.ts`; every async wrapper funnels through it. Pre-aborted signal rejects synchronously-via-microtask; post-abort removes listener cleanly via `signal.addEventListener('abort', onAbort, { once: true })`.
- **D-05 Zod boundary kept:** `CardArraySchema = z.array(CardLikeSchema.passthrough())` validates every `.next()` payload and every `Cards.byId` result before it reaches the store layer.
- **D-07 Card re-export:** `export type { Card, CardSearch } from 'scryfall-api'` — sole type import point; consumers import from `'../lib/scryfall'` / `'./scryfall'`.
- **D-09 shared throttle (library-owned):** no local throttle primitives copied over; the library's module-global 100ms spacer is trusted.
- **D-10 query-builder fallback:** all three builders moved to `src/lib/scryfall-queries.ts` with zero library coupling; ported all builder-related tests from the deleted-in-02.3-05 `scryfall-client.test.ts` block (lines 92-115 and 234-299 equivalents).

**Scope boundary respected:**
- `src/lib/scryfall-client.ts` + `.test.ts` UNCHANGED (deletion is plan 02.3-05).
- `src/lib/partner-detection.ts`, `src/types/card.ts`, components — UNCHANGED (retype is plan 02.3-03).

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Avoided `@scryfall/api-types` import in card-cache.ts**

- **Found during:** Task 3 (card-cache migration)
- **Issue:** The plan's straightforward rewrite would have needed to assign a `Card` value into `CachedCard.cardJson` (still typed as `ScryfallCard.Any` until plan 02.3-03). The obvious fix — `import type { ScryfallCard } from '@scryfall/api-types'` for a localized `as unknown as ScryfallCard.Any` cast — is rejected by the plan's acceptance grep: `grep -F "@scryfall/api-types" src/lib/card-cache.ts` must return NOTHING.
- **Fix:** Used `type CardJson = CachedCard['cardJson']` indexed-type lookup, then cast with `card as unknown as CardJson`. Same runtime behavior (TS erases the cast), zero new imports, grep clean.
- **Files modified:** `src/lib/card-cache.ts`
- **Verification:** `grep -F "@scryfall/api-types" src/lib/card-cache.ts` → no match; `vitest run src/lib/card-cache.test.ts` → 7/7 pass.
- **Committed in:** `65fb963` (Task 3)
