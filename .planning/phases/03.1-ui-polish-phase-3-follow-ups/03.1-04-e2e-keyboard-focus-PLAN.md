Focus-assertion pattern in Playwright:
```typescript
await page.keyboard.press('Tab');
await expect(locator).toBeFocused();
// Verify the focus ring is actually painted (class-level presence plus computed style):
await expect(locator).toHaveClass(/focus:ring-2/);
// OR, stricter — computed box-shadow / outline inspection via evaluate:
const hasRing = await locator.evaluate(el => {
  const cs = getComputedStyle(el);
  // Tailwind focus:ring renders as box-shadow by default.
  return cs.boxShadow !== 'none' && cs.boxShadow !== '';
});
expect(hasRing).toBe(true);
```

Scroll-reset assertion pattern:
```typescript
// Grab the outer scroll container (flex-[2] wrapper in DeckWorkspace).
// DeckWorkspace currently has no testid on that wrapper — we'll query via its
// child: the `div` that contains `[data-testid="deck-column"]`.
const outerWrapper = page.locator('[data-testid="deck-column"]').locator('..');
// Scroll it manually to simulate user scroll.
await outerWrapper.evaluate(el => { el.scrollTop = 200; });
// Click view toggle.
await page.getByRole('button', { name: 'Grid' }).click();
// Assert scrollTop reset.
const st = await outerWrapper.evaluate(el => el.scrollTop);
expect(st).toBe(0);
```
