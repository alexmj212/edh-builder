<threat_model>
- **T-05 (medium): Fixture JSON leaks real user data** — hand-crafted fixtures
  could accidentally embed a local Scryfall API key or test account email.
  Mitigation: fixtures MUST be derived only from Scryfall's public card data
  (name, mana_cost, type_line, image_uris, etc.) — no request headers, no
  session tokens, no email fields. Executor MUST `grep -i "token\|bearer\|email\|api_key"` across e2e/fixtures and confirm zero matches before committing.
- **T-06 (high if mismatched): Scryfall route matcher too broad** — a catch-all
  `page.route('**/*', ...)` matcher would intercept Vite HMR and break the app.
  Mitigation: matcher is scoped to `api.scryfall.com` exactly as the RESEARCH.md
  Pattern 1 skeleton shows. Acceptance criteria greps for the specific regex.
- **T-07 (medium): Unmatched Scryfall routes silently hit real API** — if the
  fixture's fallthrough returns `route.continue()` instead of `route.fulfill(599)`,
  a spec typo could leak real traffic. Mitigation: fallthrough MUST be
  `route.fulfill({ status: 599, body: 'stubScryfall: unhandled route ...' })`.
  Verified via grep in acceptance criteria.
- **T-08 (low): Locator regex too greedy** — `new RegExp(cardName, 'i')` with a
  name containing regex metacharacters (apostrophe, comma) could fail oddly.
  Mitigation: cards in scope (Thrasios/Tymna/Atraxa) — Atraxa has an apostrophe
  in "Praetors' Voice" but we match on "Atraxa" only, safe. Document in helper
  JSDoc.

No `high` findings that block — T-06 and T-07 are mitigated by the canonical
skeleton and grep-verifiable acceptance criteria.
</threat_model>

      reason: string;
    }

    /** Single, named allowlist per D-14. New entries REQUIRE a reason. */
    export const KNOWN_BENIGN_CONSOLE: BenignRule[] = [
      {

        textIncludes:     'Persistent storage not granted',
        reason:           'Expected browser behavior for non-installed/incognito origins (Phase 02.1 UAT test 1)',
      },
    ];

    export interface CapturedMessage { type: string; text: string; location: string }

    export function installConsoleGate(page: Page): CapturedMessage[] {
      const unexpected: CapturedMessage[] = [];
      page.on('console', (msg: ConsoleMessage) => {
        const type = msg.type();
        if (type !== 'error' && type !== 'warning') return;
        const location = `${msg.location().url}:${msg.location().lineNumber}`;
        const text = msg.text();
        const isBenign = KNOWN_BENIGN_CONSOLE.some(
          r => location.includes(r.locationIncludes) && text.includes(r.textIncludes),
        );
        if (!isBenign) unexpected.push({ type, text, location });
      });
      return unexpected;
    }
